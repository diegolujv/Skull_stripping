Customized skull-stripping MRI images (CSS) software application was built using a graphical user interface (GUI) in MATLAB by app designer that offers flexibility and precision in brain extraction from magnetic resonance imaging (MRI) scans. 

It aims to overcome the rigidity of current automatic applications by allowing the user to adapt the results to specific clinical needs. The algorithm also seeks to facilitate the detection of subtle anatomical changes over time and to ensure that the generated images are optimal for subsequent analysis using advanced techniques such as convolutional neural networks (CNNs).

Users are encouraged to reference the following article:

Luján Villarreal, D., "A Hybrid Approach for Customized Skull-Stripping MR Brain Images," 2023 8th International Conference on Signal and Image Processing (ICSIP), Wuxi, China, 2023, pp. 385-389, doi: 10.1109/ICSIP57908.2023.10270817.
